The KY-038 driver reads sound levels from a KY-038 microphone sensor module. It supports both the digital comparator output (DO pin) and the analog envelope output (AO pin).

```zig
const avr = @import("avr_zig");
const ky_038 = avr.drivers.sensor.ky_038;
```

## Types

### `ActiveLevel`

```zig
pub const ActiveLevel = enum {
    low,
    high,
};
```

Specifies which logic level on the comparator output indicates sound detection. The active level depends on the module's circuit design.

| Variant | Meaning |
|---|---|
| `.low` | Sound detected when the pin reads low |
| `.high` | Sound detected when the pin reads high |

### `DigitalState`

```zig
pub const DigitalState = enum {
    quiet,
    sound_detected,
};
```

| Variant | Meaning |
|---|---|
| `.quiet` | No sound above the comparator threshold |
| `.sound_detected` | Sound level exceeds the comparator threshold |

## Functions

### `initDigital`

```zig
pub fn initDigital(comptime pin: gpio.Pin, enable_pullup: bool) void
```

Configures the digital comparator output pin as input.

| Parameter | Type | Description |
|---|---|---|
| `pin` | `gpio.Pin` | The pin connected to the sensor's DO output (comptime) |
| `enable_pullup` | `bool` | Enable the internal pull-up resistor |

### `readDigitalRaw`

```zig
pub fn readDigitalRaw(comptime pin: gpio.Pin) bool
```

Returns the raw logic level of the comparator output pin.

| Parameter | Type | Description |
|---|---|---|
| `pin` | `gpio.Pin` | The pin connected to the sensor's DO output (comptime) |

**Returns:** `bool` -- `true` if the pin is high, `false` if low.

### `readDigital`

```zig
pub fn readDigital(comptime pin: gpio.Pin, comptime active_level: ActiveLevel) DigitalState
```

Reads the comparator output and interprets it according to the chosen active level.

| Parameter | Type | Description |
|---|---|---|
| `pin` | `gpio.Pin` | The pin connected to the sensor's DO output (comptime) |
| `active_level` | `ActiveLevel` | Which logic level indicates sound detection (comptime) |

**Returns:** `DigitalState` -- `.sound_detected` if the pin matches the active level, `.quiet` otherwise.

### `isSoundDetected`

```zig
pub fn isSoundDetected(comptime pin: gpio.Pin, comptime active_level: ActiveLevel) bool
```

Convenience function that returns `true` when sound is detected.

| Parameter | Type | Description |
|---|---|---|
| `pin` | `gpio.Pin` | The pin connected to the sensor's DO output (comptime) |
| `active_level` | `ActiveLevel` | Which logic level indicates sound detection (comptime) |

**Returns:** `bool` -- equivalent to `readDigital(pin, active_level) == .sound_detected`.

### `readAnalog`

```zig
pub fn readAnalog(comptime pin: adc.AnalogPin) u16
```

Reads the analog microphone output using the [ADC](../hal/adc/). Returns a 10-bit value (0--1023) representing the current sound envelope level.

| Parameter | Type | Description |
|---|---|---|
| `pin` | `adc.AnalogPin` | The analog pin connected to the sensor's AO output (comptime) |

**Returns:** `u16` -- the 10-bit ADC reading.

:::note
Call `avr.adc.init()` before using `readAnalog`. The digital functions (`initDigital`, `readDigital`, etc.) do not require ADC initialization.
:::

## Example

```zig
const avr = @import("avr_zig");
const gpio = avr.gpio;
const adc = avr.adc;
const uart = avr.uart;
const time = avr.time;
const ky_038 = avr.drivers.sensor.ky_038;

pub fn main() void {
    uart.init();
    time.init();
    adc.init();

    ky_038.initDigital(.D7, false);

    while (true) {
        // Digital threshold detection
        if (ky_038.isSoundDetected(.D7, .high)) {
            uart.write("Sound!\r\n");
        }

        // Analog level reading
        const level = ky_038.readAnalog(.A0);
        uart.writeInt(u16, level);
        uart.write("\r\n");

        time.sleep(100);
    }
}
```
