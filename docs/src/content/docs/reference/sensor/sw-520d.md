The SW-520D driver reads the state of a ball-type tilt switch sensor. The switch closes when tilted past its threshold angle and opens when upright.

```zig
const avr = @import("avr_zig");
const sw_520d = avr.drivers.sensor.sw_520d;
```

## Types

### `State`

```zig
pub const State = enum {
    open,
    closed,
};
```

| Variant | Meaning |
|---|---|
| `.open` | Switch is open (upright / no tilt detected) |
| `.closed` | Switch is closed (tilted past threshold) |

:::note
When the pin reads high (pulled up), the switch is open. When the pin reads low (switch conducting to ground), the switch is closed. Enable the internal pull-up or provide an external pull-up resistor.
:::

## Functions

### `init`

```zig
pub fn init(comptime pin: gpio.Pin, enable_pullup: bool) void
```

Configures the sensor pin as a digital input.

| Parameter | Type | Description |
|---|---|---|
| `pin` | `gpio.Pin` | The pin connected to the tilt switch (comptime) |
| `enable_pullup` | `bool` | Enable the internal pull-up resistor |

:::tip
Pass `true` for `enable_pullup` unless you have an external pull-up resistor on the sensor line.
:::

### `state`

```zig
pub fn state(comptime pin: gpio.Pin) State
```

Returns the current switch state.

| Parameter | Type | Description |
|---|---|---|
| `pin` | `gpio.Pin` | The pin connected to the tilt switch (comptime) |

**Returns:** `State` -- `.open` when the pin reads high, `.closed` when it reads low.

### `isOpen`

```zig
pub fn isOpen(comptime pin: gpio.Pin) bool
```

Returns `true` when the switch is open (upright).

| Parameter | Type | Description |
|---|---|---|
| `pin` | `gpio.Pin` | The pin connected to the tilt switch (comptime) |

### `isClosed`

```zig
pub fn isClosed(comptime pin: gpio.Pin) bool
```

Returns `true` when the switch is closed (tilted).

| Parameter | Type | Description |
|---|---|---|
| `pin` | `gpio.Pin` | The pin connected to the tilt switch (comptime) |

## Example

```zig
const avr = @import("avr_zig");
const uart = avr.uart;
const time = avr.time;
const sw_520d = avr.drivers.sensor.sw_520d;

pub fn main() void {
    uart.init(115200);
    time.init();

    sw_520d.init(.D5, true);

    while (true) {
        if (sw_520d.isClosed(.D5)) {
            uart.write("Tilted!\r\n");
        }
        time.sleep(200);
    }
}
```
